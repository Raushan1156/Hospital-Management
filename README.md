## Overview
Hospital Management System is a **Java Spring Boot backend service** designed to manage core hospital operations such as **secure user authentication, patient and doctor management**, and controlled access to APIs.

The project focuses on **real-world backend engineering practices**, including **JWT-based security, layered architecture, transaction management, and database integration using JPA/Hibernate**.

---

## Key Features
- JWT-based authentication and authorization
- Role-Based Access Control (RBAC)
- Secure RESTful APIs
- Patient and Doctor management
- Transaction management with rollback handling
- Centralized exception handling
- Clean layered architecture

## Security Flow
1. User signs up using the signup API
2. User logs in and receives a JWT token
3. JWT token is sent with each secured request
4. Custom JWT filter validates token before request processing

## Database Setup & Environment

### PostgreSQL (Dockerized)
- PostgreSQL runs inside a Docker container
- Container exposed on **port 9000**
- Application connects via JDBC using externalized configuration

### Database Inspection
- **DBeaver** is used to:
  - Inspect the schema generated by Hibernate
  - Validate entity relationships
  - Debug persisted data and transactions

### Configuration
- Database properties are managed via `application.yaml`
- Hibernate dialect configured for PostgreSQL
